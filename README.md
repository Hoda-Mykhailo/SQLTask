# SQLTask
This repository create for test task in ITExpert. We have 5 task with SQL requests.

Всі завдання виконанні, тільик є нюанс із 5 завданням, там в умовах потрібно знайти нагадування для HR на сьогоднішній день. В таблиці reminders дати за 03 місяць, тому при наданні запиту на виході таблиця пуста.
Якщо зробити зміни або додати нові дані в ту таблицю із датами на сьогодні, то має виводитись в таблці ці нагадування.
В цілому, на мій суб'єктивний погялд, завдання виконанні, так можливо десь не ідеально, але при запуску таблиці із даними виводятьтся, хочаб з якимись.

З нетерпінням буду чекати на фідбек по тестовому та подальшого звязку. Дякую.
